ありがとうございます。



1. チェックポイント★リベンジとは
チェックポイント★リベンジは、フリーフォントです。
和文総合書体処女作「チェックポイントフォント」をベースに、
より洗練されたデザインへと昇華させるべく、
ほぼ全ての文字/記号を一から作り直し（＝リベンジ）ています。

もちろん、独特の極太感は健在。太さのバランスも、文字種同士で
できるだけ統一するよう努めました。
本文用には多少不向きではありますが、
「ここぞ！」と強調したい場面で効果を発揮できるかもしれません。
ぜひ、様々な用途でお使いいただければと思います。



2. 収録文字
・ひらがな
・カタカナ…全角/半角
・アルファベット…全角/半角
・数字…全角/半角
・記号…ギリシア文字、キリル文字、その他一式網羅。アクサンにも対応
・漢字…JIS第一水準2,965字＋日本漢字能力検定準1級配当漢字
	＋人名用漢字（一部を除く）
	＋JIS第二水準を中心に比較的使用頻度が高いと思われる漢字
	計：約4,400字



3. 動作環境
現在のところ、
Microsoft Windows XP、Vista、7、8、10で動作を確認しております。



4. ご使用・配布について
ウェブサイトや印刷物、動画などでの使用のほか、
商用利用（同人誌などでの使用）についても、
特に制限は設けておりません。
ご自由にお使いいただければ制作者冥利に尽きます。

使用許可連絡も必要ございませんが、
よろしければ事後でも結構ですので、
web拍手やTwitterなどでご一報いただければありがたいです。

なお、字形あるいは著作権表示、
もしくはその両方を改変しての二次配布はご容赦くださいませ。



5. 免責
当フォントの使用により万が一生じた損害に対し、
当方は一切の責任を負いかねます。あらかじめご了承くださいませ。



6. その他
当フォントは、『アメリカ横断ウルトラクイズ』の二次創作物です。
著作権は、それぞれの権利所有者に帰属します。

収録されている文字・記号の種類および字形、文字幅、変換割り当て、
またダウンロード用のリンクなどは予告なく変更する場合があります。
